## ✨ 주요 기능

- 🎨 **6가지 테마** — 노랑(기본) / 파랑 / 초록 / 분홍 / 다크 / 미드나잇
- 🖱 **헤더 우클릭 메뉴** — 창에서 바로 설정 접근 (트레이와 동일)
- 🔀 **드래그 정렬** — `≡` 핸들을 드래그해 항목 순서(우선순위) 변경
- 🎨 **아이템 색상 태그** — 호버 시 `●` 버튼으로 항목별 색상 설정
- ✏️ **인라인 편집** — 호버 시 `✎` 버튼으로 해당 줄에서 바로 수정
- 📅 **미완료 항목 유지** — 날짜가 지나도 사라지지 않음 (`N일 전` 뱃지)
- 💾 **자동 저장** — 추가·체크·삭제·수정 즉시 저장
- 🗂 **시스템 트레이** — 닫기(`×`)해도 트레이로 숨겨져 백그라운드 실행
- 📌 **항상 위에 표시** — 토글 가능 (기본 OFF)
- 🔤 **글씨 크기 조절** — 9 / 10 / 11 / 12 / 13pt 선택
- 🔆 **투명도 조절** — 75% / 85% / 95% / 100%
- 💾 **백업 / 복원** — 날짜별 JSON 백업, 시작 시 자동 백업
- 🚀 **시작프로그램 등록** — 윈도우 부팅 시 자동 실행

## 🖥 UI 미리보기

```
┌─────────────────────────────────────┐  ← 3px 액센트 스트립
│ ✓ 할 일  [3]          Apr 22  Wed − ×│  ← 헤더 (드래그 이동 / 우클릭 설정)
├─────────────────────────────────────┤
│≡ ▌ ○  회의 자료 준비해서 팀장님께...  ● ✎ ×│  ← 호버 시 핸들·버튼 표시
│  ▌ ○  이번 주 보고서 제출하기         ● ✎ ×│  ← ▌ = 컬러 바
│  ▌ ●̶  이메일 답장 (어제)             ● ✎ ×│  ← 완료: ● + 취소선
├─────────────────────────────────────┤
│ 1 / 3 완료                 🗑 완료삭제│
├─────────────────────────────────────┤
│  할일을 입력하세요...            [+] │
└─────────────────────────────────────┘
```

## 🚀 실행 방법

### 방법 1 — Python으로 바로 실행

```bash
pip install PyQt6
python main.py
```

### 방법 2 — EXE 빌드 후 실행

```bash
pip install PyQt6 pyinstaller
build.bat
```

빌드 결과물: `dist/할일위젯.exe`

## ⚙️ 사용법

| 동작 | 방법 |
|------|------|
| 할일 추가 | 입력창 타이핑 후 `Enter` 또는 `+` 버튼 |
| 완료 체크/해제 | `○` / `●` 버튼 클릭 |
| 텍스트 수정 | 항목 호버 → `✎` 버튼 |
| 색상 설정 | 항목 호버 → `●` 버튼 → 팔레트 선택 |
| 순서 변경 | 항목 호버 → `≡` 핸들 드래그 |
| 항목 삭제 | 항목 호버 → `×` 버튼 |
| 완료 항목 삭제 | 하단 `🗑 완료삭제` 버튼 |
| 창 이동 | 헤더 드래그 |
| 창 크기 조절 | 우하단 `◢` 그립 드래그 |
| 창 숨기기 | `−` 또는 `×` → 트레이로 최소화 |
| 다시 열기 | 트레이 아이콘 클릭 |
| 설정 변경 | 헤더 **우클릭** 또는 트레이 아이콘 **우클릭** |

### 설정 메뉴 (헤더 / 트레이 우클릭)

- ✅ 항상 위에 표시 토글
- 🔆 투명도 조절 (75% / 85% / 95% / 100%)
- 🎨 테마 색상 (노랑★ / 파랑 / 초록 / 분홍 / 다크 / 미드나잇)
- 🔤 글씨 크기 (9 / 10 / 11 / 12 / 13pt)
- 🚀 윈도우 시작 시 자동 실행
- 💾 오늘 날짜로 백업 / 백업 파일 불러오기
- ❌ 완전 종료
